About this prototype

This frontend is a clone-style prototype of an AI-based allocation engine for the PM Internship Scheme. It demonstrates UI patterns for searching, filtering, and presenting internship recommendations with a simulated match score. The real system would include backend services, authentication, and secure data handling.

Features included in this frontend demo

  • Search & filter internships by domain, location and skills
  • Simulated AI match score computed in-browser
  • Responsive cards, detail modal, and pagination
  • Client-side "apply" flow simulation (no real submissions)

How to use

Open index.html in a browser. For a complete system, wire the frontend to backend REST APIs.
